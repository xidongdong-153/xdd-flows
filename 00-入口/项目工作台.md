## 当前输出习惯

| 项目 | 约定 |
| --- | --- |
| 回答顺序 | 先给结论，再给修补方案或使用方式 |
| 解释方式 | 用短小直接的中文，不复读背景 |
| 结构形式 | 多项内容优先用 Markdown 表格 |
| 等级标记 | 不写 `P0`、`P1` 这类等级 |
| 结束收口 | 结束时明确说明这次用到的 skill |
