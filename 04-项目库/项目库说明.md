# 项目库说明

`04-项目库` 用来放每个项目的项目卡片。

## 使用规则

1. 一个项目一张主卡片。
2. 项目卡片统一用 [[03-模板/项目卡片模板]] 创建。
3. 只有 `note_type: project` 的笔记会进入 [[04-项目库/项目总表.base]]。
4. 项目结束后，把状态改成 `归档`，再移到 `90-归档`。

## 字段提醒

- `status`：看当前项目是否在推进
- `stage`：看当前推进到哪一步
- `doc_status`：看仓库文档是否已经补齐
- `agent_ready`：看是否已经整理到能直接给 Agent 读

## 建议做法

- 项目卡片里优先维护固定入口，不要把零散记录堆得太散。
- 真的需要展开记录时，再给项目单独建子笔记。
